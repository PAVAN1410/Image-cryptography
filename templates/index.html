RubImg is a web application aiming to encrypt your images using a very powerful rubix cube algorithm. It also helps the other end user who recieves image to decrypt the image with just 2 clicks

    Automated encryption using rubix cube algorithm

Usage

Encryption

Click Encryption menu

Click on choose file

Upload image

Click on Encrypt Image

Take the zip file of encrypted Images and some other crucial information

Decryption

Click Decryption menu

Click on choose file

Upload zip file

Click on decrypt image

Take the Secret image that you uploaded as a encrypted image
